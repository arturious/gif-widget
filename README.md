# Desktop GIF & Image Widget for macOS

Легкий и быстрый нативный виджет для отображения GIF-анимаций и картинок на рабочем столе macOS. Написан на Swift/AppKit, работает в один процесс для всех окон, потребляет минимум оперативной памяти и ресурсов процессора. Умеет работать в фоновом режиме и скрыт из Dock.

## Как запустить

Вы можете запустить скомпилированное приложение прямо из терминала:

```bash
# Перейдите в папку проекта
cd /Users/arturbakhtygereyev/dev/gif-widget

# Запустите виджет
./gifwidget
```

*При первом запуске откроется стандартный диалог выбора файла. Виджет запомнит ваш выбор и при следующем запуске откроет его автоматически.*

## Возможности (Управление виджетом)

### 1. Интеграция с рабочим столом (Locked Mode)
Когда вы блокируете виджет:
- Он опускается на самый нижний слой обоев (**под иконки и файлы рабочего стола**). Вы можете размещать любые файлы поверх виджета.
- Он становится полностью **клик-сквозным** (все клики проходят сквозь виджет прямо к файлам на рабочем столе или Finder).

### 2. Взаимодействие (Cmd + Click / Cmd + Right-click)
Поскольку заблокированный виджет находится под иконками Finder и не кликабелен, для взаимодействия с ним:
- **Зажмите клавишу Command (Cmd / ⌘)** и наведите мышь на виджет. Он автоматически приподнимется над иконками рабочего стола.
- Продолжая удерживать `Cmd`, вы можете:
  - **Перетащить виджет** левой кнопкой мыши.
  - Нажать **правой кнопкой мыши**, чтобы открыть контекстное меню.
- Как только вы отпустите `Cmd` (и кнопку мыши), виджет мгновенно опустится обратно под иконки рабочего стола.

### 3. Контекстное меню виджета
В меню доступны следующие пункты:
- **Lock / Unlock** — Блокировать/разблокировать конкретный виджет.
- **Choose image/GIF...** — Сменить файл изображения.
- **New Widget** — Создать новое независимое окно виджета.
- **Size** — Масштабировать виджет (Increase/Decrease/Reset).
- **Opacity** — Настроить прозрачность от 100% до 20%.
- **Quit** — Закрыть именно этот виджет (удаляет его из автозапуска).
- **Quit All** — Закрыть все виджеты и завершить приложение (сохраняя их в памяти для следующего запуска).

## Сохранение настроек (Persistence)

Виджет автоматически запоминает и восстанавливает при следующем запуске:
*   Путь к выбранному файлу (для каждого виджета отдельно).
*   Положение и размер виджета на экране.
*   Режим прозрачности (Opacity).
*   Состояние блокировки (Lock state).
*   Количество и состав активных окон (после «Quit All» все виджеты откроются снова при перезапуске приложения!).
