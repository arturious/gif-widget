# Desktop GIF & Image Widget for macOS

Легкий и быстрый нативный виджет для отображения GIF-анимаций и картинок на рабочем столе macOS. Написан на Swift/AppKit, потребляет минимум оперативной памяти и ресурсов процессора. Умеет работать в фоновом режиме, скрыт из Dock и управляется через строку меню (Menu Bar).

## Как запустить

Вы можете запустить скомпилированное приложение прямо из терминала:

```bash
# Перейдите в папку проекта
cd /Users/arturbakhtygereyev/dev/gif-widget

# Запустите виджет
./gifwidget
```

*При первом запуске откроется стандартный диалог выбора файла. Виджет запомнит ваш выбор и при следующем запуске откроет его автоматически.*

## Возможности (Управление виджетом)

### 1. Интеграция с рабочим столом (Locked Mode)
Когда вы блокируете виджет:
- Он опускается на самый нижний слой обоев (**под иконки и файлы рабочего стола**). Вы можете размещать любые файлы поверх виджета.
- Он становится полностью **клик-сквозным** (все клики проходят сквозь виджет прямо к файлам на рабочем столе или Finder).

### 2. Иконка в Строке меню (Menu Bar)
Так как заблокированный виджет находится под иконками Finder и не кликабелен, для управления им в верхнем меню macOS появляется специальная иконка (в виде стопки картинок 🖼️):
Нажав на неё, вы получите полный контроль над всеми запущенными виджетами:
- **New Widget** — Создать новое независимое окно виджета.
- **Список активных виджетов** (по имени файла). Нажав на любой из них, вы можете:
  - **Lock / Unlock** — Блокировать/разблокировать конкретный виджет. При разблокировке он поднимается над файлами рабочего стола и становится кликабельным (его можно перетаскивать).
  - **Choose image/GIF...** — Сменить файл изображения.
  - **Size** — Масштабировать конкретный виджет (Increase/Decrease/Reset).
  - **Opacity** — Настроить прозрачность от 100% до 20%.
  - **Close Widget** — Закрыть именно этот виджет.
- **Quit gifwidget** — Полностью выйти из приложения и закрыть все виджеты.

## Сохранение настроек (Persistence)

Виджет автоматически запоминает и восстанавливает при следующем запуске:
*   Путь к выбранному файлу (для каждого виджета отдельно).
*   Положение и размер виджета на экране.
*   Режим прозрачности (Opacity).
*   Состояние блокировки (Lock state).
