# Desktop GIF & Image Widget for macOS

Легкий и быстрый нативный виджет для отображения GIF-анимаций и картинок на рабочем столе macOS. Написан на Swift/AppKit, потребляет минимум оперативной памяти и ресурсов процессора. Умеет работать в фоновом режиме и скрыт из Dock.

## Как запустить

Вы можете запустить скомпилированное приложение прямо из терминала:

```bash
# Перейдите в папку проекта
cd /Users/arturbakhtygereyev/dev/gif-widget

# Запустите виджет
./gifwidget
```

*При первом запуске откроется стандартный диалог выбора файла. Выберите любую гифку или картинку. Виджет запомнит ваш выбор и при следующем запуске откроет его автоматически.*

## Возможности (Управление виджетом)

Нажмите **правой кнопкой мыши** по виджету, чтобы открыть контекстное меню (на английском языке):

1. **Lock Widget (Cmd+Click to unlock/drag)** — Переводит виджет в заблокированный режим:
   - Перемещает его на слой обоев (**Below Windows**).
   - Делает его **клик-сквозным** (все клики будут проходить сквозь виджет к Finder или другим окнам).
   - Чтобы взаимодействовать с виджетом в этом режиме (перетащить или открыть меню для разблокировки), **зажмите клавишу Command (Cmd / ⌘)** и кликните (или перетащите).
2. **Choose Image/GIF...** — Сменить текущий GIF/картинку.
3. **Window Position** (Недоступно в режиме Lock):
   - **Always on Top** — Виджет всегда будет поверх остальных окон.
   - **Normal Window** — Обычное окно.
   - **Below Windows (Desktop)** — Расположить на рабочем столе под всеми окнами.
4. **Opacity** — Прозрачность виджета от 100% до 20%.
5. **Quit Widget** — Выход из приложения.

---

## Сохранение настроек (Persistence)

Виджет автоматически запоминает и восстанавливает при следующем запуске:
*   Путь к выбранному файлу (изображению или GIF).
*   Положение и размер виджета на экране.
*   Режим прозрачности (Opacity).
*   Состояние блокировки (Lock state).
